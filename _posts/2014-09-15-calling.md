### Make a Call

**Step 1.** Implement `CallStateListener` to receive call state changes in case if call is failed or remote ended the call without answering.

```java
@Override
public void onCallEnded() {
	
}

@Override
public void onCallFailed() {

}
```

**Step 2.** Register `InCallActivity` to your Manifest.xml

```xml
<activity android:name="com.bit6.sdk.ui.InCallActivity"
    android:label="@string/app_name"
    android:configChanges="orientation|screenSize"
    android:theme="@android:style/Theme.Black.NoTitleBar.Fullscreen">
</activity> 
```

To make a voice or video call to a destination:

```java
Address to = Address.parse("usr:john");
bit6.startCall(to, isVideo);
```


### Receive a Call

**Step 1.** Register Broadcast Receiver for `com.bit6.intent.call` intent in your Manifest.xml

```xml
<receiver android:name=".IncomingCallReceiver" android:enabled="true">
    <intent-filter>
        <action android:name="com.bit6.intent.call"></action>
    </intent-filter>
</receiver>
```

**Step 2.** Create a receiver class which extends BroadcastReceiver, get the Call object and send it to you activity that handles incoming calls.

```java
public class IncomingCallReceiver extends BroadcastReceiver{

	@Override
	public void onReceive(Context context, Intent intent) {
		Intent i = new Intent(context, IncomingCallActivity.class);
		i.setFlags(Intent.FLAG_ACTIVITY_NEW_TASK);
		i.putExtra("call", intent.getExtras().getParcelable("call"));
		context.startActivity(i);
	}
}
```

**Step 3.** Create an activity to handle incoming calls.

```java
// Get Call object from the intent
Call call = getIntent().getExtras().getParcelable("call");

// Caller information
String caller = call.getSender();

// Is this a voice or video call?
boolean isVideo = call.isVideo();

// Either answer the call
bit6.answerCall(call);

// Or reject the call
bit6.rejectCall(call);

```
